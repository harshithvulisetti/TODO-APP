INSTRUCTIONS

  • This is an Todo event handler website
  • TO add an event, you have to enter the event name and select date and time when the event is planned and click on add
  • If the event is completed you can delete or mark the event as completed
  • if you delete the task or event it will be erased
  • If you mark the event as completed the event will be moved to completed events list
  • If you click on the completed tasks button, it will show the completed tasks with underline along with the active tasks. To distingush between active and completed events we added an underline to completed task.
